\subsection{can と be able to}
    \fitblankbf[]{can} ＝ ( \fitblankbfshort{be} )( \fitblankbfshort{able} )( \fitblankbfshort{to} ) 『 ～する ことができる 』

\subsection{can の過去形}
    \fitblankbf[]{can} の過去形は( \fitblankbf[]{could} )である。

    ※ \fitblankbf[]{will} と \fitblankbf[]{can} は似ている。覚えるときはまとめて覚えよう！！

    \begin{theorembox}{\textbf{will と can の対応}}
        \begin{itemize}
            \item \fitblankbf[]{will} ＝ be going to ＋ 動詞の原形 $\Longrightarrow$ \fitblankbf[]{would}
            \item \fitblankbf[]{can} ＝ be able to ＋ 動詞の原形 $\Longrightarrow$ \fitblankbf[]{could}
        \end{itemize}
    \end{theorembox}
    \vspace{0.5em}

    ※上記の \fitblankbf[]{be} は( \fitblankbf[]{be 動詞} )という意味であるので、主語の( \fitblankbf[]{単複} )や( \fitblankbf[]{時制} )によって変化させなければならない。

    \begin{enumerate}
        \item She can swim very fast.
        \dialogueblank{彼女はとても速く泳ぐことができる。}
        \vspace{0.5em}
        \item I am able to speak a little English.
        \dialogueblank{私は少し英語を話すことができる。}
        \vspace{0.5em}
        \item He could run ten kilometers when he was young.
        \dialogueblank{彼は若いころ10キロ走ることができた。}
    \end{enumerate}

\subsection{may}
    \fitblankbf[]{may} 『～ かもしれない ( 推定 )、 ～ してもよい ( 許可 )』

    \fitblankbf[]{May I} ～？ 『～ していいですか ？』

    \begin{enumerate}
        \item It may rain this afternoon.
        \dialogueblank{今日の午後は雨が降るかもしれない。}
        \vspace{0.5em}
        \item May I use your pen?
        \dialogueblank{あなたのペンを使ってもいいですか。}
    \end{enumerate}

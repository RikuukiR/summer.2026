\section{講習の目的}
  \subsection{講習全体を通しての目的}
    \begin{enumerate}
      \item 公立高校入試の問題形式や単元別出題割合などについて知ってもらう
      \item 別紙の自己分析シートを使用した自己分析をしてもらう
      \item 公立高校入試や北海道コンクールの問題形式に慣れてもらう
      \item 受験生としての自覚も持ってもらう
    \end{enumerate}

  \subsection{数学の講習での目的}
    \begin{enumerate}
      \item 基礎計算能力を養ってもらう
      \item 文章問題の立式に慣れてもらう
      \item 文字を使った証明問題に慣れてもらう
      \item 作図問題に慣れてもらう
      \item 平面図形の証明問題に慣れてもらう
      \item 自分の苦手な単元を見つけてもらう
    \end{enumerate}

  \subsection{英語の講習での目的}
    \begin{enumerate}
      \item リスニングに慣れてもらう
      \item 基本的な語彙力を養ってもらう
      \item 英作文に慣れてもらう
      \item 頻出の文法を身につけてもらう
      \item 長文読解に慣れてもらう
      \item 自分の苦手な文法を見つけてもらう
    \end{enumerate}

\section{講習の内容}
  \subsection{数学}
    \begin{enumerate}
      \item 基礎計算問題
      \item 一次方程式の利用、連立方程式の利用（文章問題の立式）
      \item 式の計算の利用（文字を用いた証明）
      \item 作図
      \item 三角形の合同、二等辺三角形の性質、平行四辺形の性質（図形の証明問題）
      \item 公立高校入試の過去問を用いた演習
      \item 北海道コンクールの過去問を用いた演習
    \end{enumerate}

  \subsection{英語}
    \begin{enumerate}
      \item リスニング
      \item 単語、イディオム、慣用表現
      \item 英作文の考え方
      \item be動詞と一般動詞の使い分け
      \item 疑問詞を含む文の疑問文
      \item 助動詞
      \item 比較
      \item 長文読解の考え方
      \item 公立高校入試の過去問を用いた演習
      \item 北海道コンクールの過去問を用いた演習
    \end{enumerate}

\section{講習の注意点}
    \begin{itemize}
      \item 2回目以降の授業の初めには、毎回確認テストを実施します。
      \item 講習全体を通してアウトプットが中心のため、体感時間は実際よりも短くなります。
      \item 講習全体を通して、公立高校入試及び北海道コンクールで各自の目標点を達成することに重点を置き、授業を展開します。
      \item 講習全体を通して、受験生としての自覚を持てるよう、指導します。
      \item 講習全体を通して、講習後も継続できるような自分なりの学習方法を確立してもらいます。
      \item 宿題は毎授業、課します。
    \end{itemize}

  \subsection{\textbf{ちょっと小話をしよう}}
  「分かる」とはそもそもどういうことだろうか。
  よく\textbf{「分かる」}とは\textbf{「分ける」}ことだと言われる。みんなの頭の中に沢山のラベルがつけられた「箱」がある
  とイメージしてみて欲しい。みんなは郵便局員のように、入ってくる情報に対して、
  これはこっちの箱、これはこっちの箱と仕分けをしていく。
  新しい情報を頭の中のしかるべき「箱」に入れることが「分かる」ことであり、
  どの箱にいれるべきかを的確にガイドしてくれる説明がいわゆる\textbf{「分かりやすい説明」}というわけである。

\medskip
\medskip
  しかし、もしやってきた情報が、みんなにとって全く新しい概念であったらどうだろうか。
  それはみんなの持っているどの箱にも分類できないようなものである。「分けられない」のだから「分からない」。
  その「分からない」状態でいることに我々は我慢できなくなってしまうのである。
  そんなとき、その概念を手近な箱に無理やり押し込んでくれる説明があると、
  それを\textbf{「分かりやすい」}と感じてしまいがちである。
  でもそれはみんなを安心させるための\textbf{その場しのぎの説明}。
  確かに分かったつもりにはなれるかもしれないが、誤った箱に入れられた情報は二度と顧みられることはなく、
  本質的な理解から永遠に遠ざかったままになってしまう。
  これが「分かりやすさ」の\uwave{とても危険な落とし穴}である。

\medskip
\medskip
  学習においてとても大切な姿勢は何か。それは「分からないを分からないものとして一旦受け入れるということ」
  だと私は思う。言ってみれば\textbf{「分からない」というラベルのついた「箱」を作り、行き場のない情報はひとまずそこに放り込んでおく}
  ということである。もちろんそれは少し居心地が悪いことかもしれない。
  でもその違和感をうまく飼い慣らしながら、ときどきは箱の中をチラチラと眺めながら先に進んでいくと、
  不思議なもので「分からない」の箱にいれておいたいくつかの情報が結びつき、
  きちんと意味を持ちはじめる日が必ず来る。頭の中にその概念をおさめる新しい「箱」が立ち上がる。
  これが真の\textbf{「わかった！」}の瞬間である。
  \uwave{「分からない」は学びの障害物ではない}。
  むしろ学びにおいてなくてはならない重要な通過点である。

\medskip
\medskip

  私の授業は「分かりやすい」授業ではないと思っている。学習自体がそもそも
  「分かりにくい」ものであり、しっかりと思考し、言語化できることを目標にしているからである。
  そこから変に逃げたりせず、「分かりにくい」事柄は「分かりにくい」ままで、
  その代わり丁寧に言葉を尽くして説明していくというスタンスを貫くことにしている。
  とりわけ日々の学習内容は新しい概念のデパートであり、初めての情報が波のようにみんなの頭の中に押し寄せてくることになる。
  どう処理していいのか分からない情報に飲み込まれそうになったときに、思い出して欲しいシンプルな「学びの3箇条」。
  \begin{itemize}
    \item \textbf{最初からすべてを分かろうとしないこと}
    \item \textbf{分からなくても，どんどん先に進むこと}
    \item \textbf{分からない部分はすぐに一度（そして何度も）質問すること}
  \end{itemize}
  大切なことは「分からない」を怖がらないことである。何かにひっかかるのはみんなが新しいことを学びつつある証
  \textbf{「分からない」こそが真の理解の入り口}なのである。
  それでは、どうぞみなさまの頭の中に「分からない」箱のご用意を。
